# Raylib
**raylib is a simple and easy-to-use library to enjoy videogames programming.**

## Version
The included raylib version is ***raylib v5.5***

## Original source
https://github.com/raysan5/raylib/releases/tag/5.5

## Code provided
The code provided in this folder is a modified, adapted version of the
original source code. (With the exception of *raygl.h*)
- *raylib.hcw* &larr; [include/raylib.h](https://github.com/raysan5/raylib/blob/master/src/raylib.h)
- *raymath.hcw* &larr; [include/raymath.h](https://github.com/raysan5/raylib/blob/master/src/raymath.h)
